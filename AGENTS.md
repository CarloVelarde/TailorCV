AGENTS
======

Purpose
-------
This file summarizes expectations for contributors and agents working on this repo.

CI
--
- R Run `ruff check .` and `ruff format --check .` to lint and format code.
- CI should fail if Ruff formatting or linting fails.

Project context
---------------
- Keep `docs/VISION.md` and `docs/ROADMAP.md` in mind during development.
- If plans change, update those docs and confirm with the project owner.
